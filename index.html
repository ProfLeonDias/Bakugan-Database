BakuganDB- The Bakugan Database is a site dedicated to help document every official release of Bakugan and their supporting accessory pieces. This site does NOT contain information on products such as McDonalds Bakugan, Morph-Lites, Keychains, or any TCG related cards.


If you want to learn about the history of the franchise, or you are a collector looking for an up-to-date resource, then this is the site for you! Whether you grew up with the original, or discovered the series through its newer iterations, you are appreciated and a welcome part of the community.


Visit our About page if you would like to learn more about the inspiration behind this project.
If you would like to contribute any additional documentation, or would like to provide higher quality images, feel free to contact the owner via the various links on the bottom of each page.
